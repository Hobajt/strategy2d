scripts for spritesheet color editing - for proper color cycling (faction color coding)

    - modify_colors.py
        - to override the colors & store result in new file
        - which colors to change comes either from color_samples.png or an array in the script
    - test_colors.py
        - to validate that cycling works
    - highlight_colors.py
        - highlights all pixels with given colors in given image
    - print_colors.py
        - prints out all the unique colors in the image
